Reemplazar imágenes

Las imágenes actuales son placeholders de Unsplash. Para reemplazarlas:

  • Guarda tus imágenes en la carpeta assets/
  • Actualiza las rutas en cada archivo HTML
  • Mantén los nombres descriptivos para facilitar el mantenimiento
  • Recomendado: imágenes optimizadas para web (max 1200px de ancho)

Personalizar datos

Edita el archivo script.js para:

  • Cambiar el número de WhatsApp (variable WHATSAPP_NUMBER)
  • Modificar la lista de servicios (variable SERVICES)
  • Actualizar textos y placeholders según necesites

Subir a GitHub Pages

  1. Crea un repositorio en GitHub
  2. Sube todos los archivos manteniendo la estructura
  3. Ve a Settings > Pages
  4. Selecciona la rama main y carpeta root
  5. Espera unos minutos y tu sitio estará online
